    3. Managed Cloud
      4.

      While the religious debates rage on between the efficacies of private cloud versus public cloud computing, a logical middle ground has emerged; the manager cloud. Managed cloud is an integrated cloud computing service which makes efficient utilization of both private and public cloud resources to perform distinct functions within the same organization. All cloud computing services should offer certain efficiencies to differing degrees, but public cloud computing services are more likely to be scalable than private clouds. Therefore, an organization can maximize its efficiency by employing public cloud services for all non-sensitive operations, only relying on a private cloud where it is required and ensuring that all of their platforms are seamlessly integrated. Technologies like OpenStack are enabling cost effective private cloud infrastructures for deployment inside organizations or managed service providers; which can be coupled with public cloud platforms like Google Cloud Platform to get the best of both worlds.
